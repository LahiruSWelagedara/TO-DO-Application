# Todo App - Frontend

An elegant and highly responsive frontend Todo application, developed using React, Vite, and Tailwind CSS. This application boasts an elegant glassmorphism design along with cool transitions.

## Features
- Modern Glassmorphism UI
- Fully Responsive Design
- Elegant Colors and Typography
- Quick Development & Build with Vite
- Task Management System (Add, Update, Delete, Mark As Done)
- Separate View For Active And Completed Tasks
- Form Validations And Errors

## Tech Stack
- **React 19**
- **Vite**
- **Tailwind CSS 4**
- **Lucide React** (Icons)

## Requirements
- Node.js (Version 18 or above)
- npm / yarn

## Setup
1. First, install dependencies.
   ```bash
   npm install
   ```
2. Then, start the development server.
   ```bash
   npm run dev
   ```
3. Finally, head over to `http://localhost:5173` (Or the provided port by Vite) in your browser.

## Connect To Backend
This frontend application automatically connects to `http://localhost:5000` backend application API. Please make sure you have both front and backend applications running.
